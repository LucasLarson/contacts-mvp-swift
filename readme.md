# Contacts

### Description
*Contacts* is an iOS app based on __MVP (Model View Presenter)__ software architectural pattern.

### Run Requirements

* Xcode 10.2.1
* Swift 5.0

### Setup

* Run `open Contacts.xcodeproj`

### NOTE: Project built using minimum or almost no third party library dependency.
